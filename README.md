# Password Generator

Простой генератор паролей, написанный на Go.

## Описание

Это консольное приложение, которое генерирует случайные пароли заданной длины. Программа запрашивает у пользователя желаемую длину пароля и создает последовательность случайных чисел.

## Установка и запуск

### Требования
- Go версии 1.25.1 или выше

### Сборка и запуск
```bash
# Клонирование репозитория
git clone <repository-url>
cd passwordGenerator

# Запуск программы
go run main.go
```

Или скомпилируйте и запустите:
```bash
go build -o password-generator
./password-generator
```

## Использование

1. Запустите программу
2. Введите желаемую длину пароля (целое число больше 0)
3. Программа сгенерирует и выведет последовательность случайных чисел указанной длины

### Пример работы
```
Какая длина пароля? Введите:
12
[1 7 3 9 0 5 2 8 6 4 1 7]
```

## Особенности реализации

- Использует криптографически безопасный генератор случайных чисел (`crypto/rand`)
- Проверяет ввод на корректность (длина должна быть больше 0)
- Генерирует пароль как последовательность цифр от 0 до 9
